# 🚀 MarketMind AI

AI-Powered Stock Decision Engine for Retail Investors

---

## 🧠 Problem

Retail investors often face:
- Information overload
- No clear buy/sell signals
- Dependence on unreliable tips

---

## 💡 Solution

MarketMind AI converts raw financial signals into actionable decisions using a structured AI scoring engine.

---

## ⚙️ Features

- Buy / Sell / Hold Signal
- Confidence Score
- Risk Detection
- AI Score (0–100)
- Buy Probability
- Market Insight Generator
- Trend Visualization

---

## 🏗️ Tech Stack

Frontend:
- HTML, CSS, JavaScript
- Chart.js

Backend:
- Node.js
- Express

---

## 🔍 How It Works

Score = (Sentiment × 10) − (Volatility × 5) + Randomness

Based on score:
- Signal generated
- Risk calculated
- Confidence derived

---

## 🚀 Run Locally

Backend:
cd backend
npm install
node server.js

Frontend:
cd frontend
npx live-server
